# CRUD de Alunos - Treino SAEP

Este projeto foi criado para praticar um CRUD completo usando apenas:

- HTML
- CSS
- JavaScript
- localStorage

## O que o projeto faz

- Cadastra alunos
- Lista os alunos em tabela
- Pesquisa por nome ou curso
- Edita um aluno existente
- Exclui um aluno
- Salva os dados no navegador

## Estrutura do CRUD

- `Create`: cadastrar aluno
- `Read`: listar alunos
- `Update`: editar aluno
- `Delete`: excluir aluno

## Arquivos

- `index.html`: estrutura da tela
- `style.css`: visual da aplicacao
- `script.js`: regras do CRUD

## Como testar

1. Abra o arquivo `index.html` no navegador.
2. Cadastre alguns alunos.
3. Teste editar, excluir e pesquisar.
4. Recarregue a pagina para ver os dados salvos.

## Como apresentar no SAEP

Se te perguntarem o funcionamento, voce pode explicar assim:

- O formulario captura os dados do aluno.
- O JavaScript valida as informacoes.
- Os dados sao guardados em um array.
- Esse array e salvo no `localStorage`.
- A tabela e renderizada na tela com os botoes de editar e excluir.
