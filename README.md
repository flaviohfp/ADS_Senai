# 💻 Curso Técnico em Análise e Desenvolvimento de Sistemas — SENAI (Turma V2)

Este repositório reúne **todos os conteúdos, projetos, exercícios e anotações** desenvolvidos ao longo do curso técnico de **Análise e Desenvolvimento de Sistemas (ADS)** da **Turma V2** do **SENAI**.

O objetivo é manter um espaço organizado e centralizado com tudo o que foi aprendido durante o curso — desde os primeiros algoritmos até os projetos integradores e aplicações completas.

---

## 🧱 Estrutura do Curso

### 🧠 1º Módulo — Fundamentos da Programação
- Lógica de Programação  
- Algoritmos e Fluxogramas  
- Linguagem C

- Pensamento Computacional  

### ⚙️ 2º Módulo — Desenvolvimento e Banco de Dados

### 🌐 3º Módulo — Integração e Projetos
- APIs e Integração de Sistemas  
- Programação Web Avançada  
- Projeto Integrador  
- Testes, Versionamento e Deploy  
- Empreendedorismo e Ética Profissional

## 🧑‍🏫 Sobre o Curso

O curso técnico de **Análise e Desenvolvimento de Sistemas (ADS)** do **SENAI** tem como objetivo formar profissionais aptos a:

- Analisar, projetar e desenvolver sistemas computacionais;  
- Aplicar boas práticas de lógica e programação estruturada;  
- Trabalhar com linguagens de alto nível, bancos de dados e versionamento de código;  
- Entender a importância da documentação e do trabalho em equipe;  
- Atuar com ética, responsabilidade e foco em soluções tecnológicas inovadoras.

## 🎯 Objetivo do Repositório

> Criar um ambiente de aprendizado contínuo e organizado, que registre a jornada completa no curso de ADS — desde os primeiros códigos até os projetos finais.

Além de servir como material de estudo e revisão, este repositório também funciona como **portfólio pessoal**, mostrando a evolução das habilidades técnicas e práticas desenvolvidas durante o curso.

---

## 📚 Disciplinas em Destaque

- **Lógica de Programação:** base do raciocínio computacional e estrutura de pensamento lógico.  
- **Programação Orientada a Objetos:** organização e reutilização de código em Java.  
- **Banco de Dados:** modelagem, normalização e consultas SQL.  
- **Desenvolvimento Web:** fundamentos de front-end e integração com back-end.  
- **Projeto Integrador:** união de todas as disciplinas em um sistema completo.
